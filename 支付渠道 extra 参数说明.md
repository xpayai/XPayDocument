# 支付渠道 extra 参数说明

特定渠道发起交易时需要的额外参数，以及部分渠道支付成功返回的额外参数。huanxun_ali_scan ，huanxun_wx_scan等 渠道不需要 extra 参数。

### pc银联支付

| extra 参数 | 说明 |
| ------ | ---- |
| callback_url | 支付成功的回调地址，在支付成功后浏览器自动跳转到指定的页面 |
| bank_code | 银行编码 详情参考银行编码参数说明 |
